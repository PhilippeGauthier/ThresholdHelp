Tagging

A tag is a descriptor that will help you locate your photos in the future. You can add tags in Dynamic View and in Photo Overlay. These are used primarily to view and sort photos in the Gallery View.

How to Use Tags

Tags are pieces of information that can be attached to a photo that act as a label. This kind of data can help make each photo unique and can help you find specific photos down the road.

Where to start

By selecting a plan type in Plan Management, you're adding a tag to be used in Dynamic View. The plan types available in Plan Management are Fixed Tags.

Types of Tags

The most common types of tags are fixed tags.

Fixed Tags

We've created nine tags for you to select from. These nine tags cover the basic types of plan sheets that are most commonly used in construction. The most common tag is Architectural.

Custom Tags

Custom tags allow you to create your own labels. These can be used in the Gallery View to search for specific photos or attributes. Examples of custom tags are Sprinkler Systems, HVAC, Seismic, etc. There's no limit to the number of custom tags you can have.

Where to Use Tags

Threshold makes extensive use of the tagging system. You'll do the majority of your tagging in Dynamic View.

Dynamic View

The tagging process starts when a photo is added to a room in Dynamic View. This is the only way to add your first fixed tag. We do this because when photos are first uploaded, they are only associated with a project. When you drag and drop a photo in Dynamic View you're adding a set (floor), room, and trade (plan type) tag.

Example: If you add a photo to the mechanical plan, the photo's first tag will be "Mechanical".

At first this photo will reside only on the mechanical plan. In the Photo Overlay you can assign multiple tags, which will allow your photo to be viewed on multiple plan types.

Photo Viewer

After you've added a photo to a room all of your secondary tagging will take place in the Photo Viewer. To add tags to a photo click on the fixed or custom tags on the lower right. When a tag has been selected it will become highlighted.

Gallery View

Gallery View is where you can use tags to find specific photos. When you select a tag on the lower left, the photos on the right will be sorted to only include photos with that specific tag. The tags, in conjunction with filtering by location and date will allow you find specific photos.
